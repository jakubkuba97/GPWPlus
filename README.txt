
This program is intended to help organize and track different firms and corporations on the polish stock market site: "gpw.pl".
It is highly customizable and was created to solidify the knowledge about Python skills such as:
	- web searching
	- exception handling
	- simple text editing
	- file management
	- frameworks basics
	- data management
	- input management
This program is not properly customized for storing and managing large amounts of data
The latest version of this product was compiled on 28.05.2019
To see this program's requirements document, please navigate to "Test GPWPlus" project

Note: Some known bugs have been purposely left out for the sake of testing this application in another project - see above
GIT support: GIT support has been added post-production and this project was not created with its use
